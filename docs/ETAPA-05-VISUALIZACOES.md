# Etapa 05 — Visualizações

As cinco visualizações utilizam os mesmos registros:

1. Tabela — edição rápida, seleção, propriedades, ordem e largura das colunas.
2. Lista — leitura compacta e subatividades recolhíveis.
3. Quadro — agrupamento por status, responsável, projeto ou prioridade.
4. Calendário — datas de início e prazo.
5. Linha do tempo — período entre início e fim e área para registros sem período.

## Visualizações salvas

Cada usuário pode salvar nome, tipo, filtros, ordenação, agrupamento, propriedades visíveis, ordem, larguras, paginação e indicação de visualização padrão. RLS restringe os registros ao próprio usuário.
