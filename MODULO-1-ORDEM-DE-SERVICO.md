# Módulo 1 — Ordem de Serviço completa

## Recursos incluídos
- Aba de materiais estruturados com quantidade, unidade, largura e situação.
- Aba de arquivos para arte, documentos e fotos, com Supabase Storage privado.
- Checklist operacional por OS, com modelo padrão da Publicolor.
- Integração com resumo, produção, instalação, histórico e comentários existentes.
- Permissões respeitando administrador, gerente, produção e visualização.

## Instalação
1. Execute no SQL Editor do Supabase:
   `supabase/migrations/20260714010000_order_workspace_module.sql`
2. Copie os arquivos atualizados para o projeto.
3. Execute `pnpm install` e `pnpm dev`.
4. Teste materiais, arquivos e checklist antes de publicar.
